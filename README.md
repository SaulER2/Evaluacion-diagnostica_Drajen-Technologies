##Ejercicios diagnósticos
Se presentan los 6 ejercicios:
1.- Políndromos
2.- Cálculo de edad exacta
3.- Generador de contraseñas
4.- Juego de ahorcado
5.- Fibonacci
6.- CRUD de usuarios

-Se requiere mínimo la version 21 de java

Si no funciona abrir el archivo con doble click:
-En la carpeta donde se encuentra el .jar escribir lo siguiente:
java -jar Evaluacion-Diagnostica-1.0-SNAPSHOT.jar
